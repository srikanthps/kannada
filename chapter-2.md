# What Are Swaragalu?

**Swaragalu (ಸ್ವರಗಳು)** are the **vowels** of the Kannada language.

A vowel can be pronounced by itself without the help of another letter.

For example:

- ಅ
- ಆ
- ಇ

Each of these letters can stand alone as an independent sound.

Later in this book, you will learn how vowels combine with consonants to form complete syllables.

# Kannada Vowels

The Kannada script has the following commonly used independent vowels.

| Kannada | Transliteration | Pronunciation |
|----------|-----------------|---------------|
| ಅ | a | as in *about* |
| ಆ | aa | as in *father* |
| ಇ | i | as in *sit* |
| ಈ | ii | as in *see* |
| ಉ | u | as in *put* |
| ಊ | uu | as in *food* |
| ಋ | ru | vocalic *r* |
| ಎ | e | as in *bed* |
| ಏ | ee | as in *they* |
| ಐ | ai | as in *aisle* |
| ಒ | o | as in *go* (shorter) |
| ಓ | oo | as in *go* |
| ಔ | au | as in *cow* |
| ಅಂ | am | anusvara |
| ಅಃ | ah | visarga |

# Reading Practice

Read the vowels slowly several times.

```text
ಅ ಆ ಇ ಈ ಉ ಊ ಋ

ಎ ಏ ಐ

ಒ ಓ ಔ

ಅಂ ಅಃ
```

Don't worry about speed.

Focus on recognizing each symbol.

# Short and Long Vowels

Some Kannada vowels occur in pairs.

| Short | Long |
|--------|------|
| ಅ | ಆ |
| ಇ | ಈ |
| ಉ | ಊ |
| ಎ | ಏ |
| ಒ | ಓ |

The long vowel is held slightly longer while speaking.

Compare:

- ಅ → ಆ
- ಇ → ಈ
- ಉ → ಊ

With practice, the difference becomes natural.

---

# Tips for Learning

Instead of trying to memorize all the vowels at once:

- Learn three or four each day.
- Read them aloud.
- Write each one several times.
- Review yesterday's vowels before learning new ones.

Regular practice is far more effective than memorizing everything in one session.

---

# Common Mistakes

## Confusing Short and Long Vowels

Beginners often confuse:

- ಅ and ಆ
- ಇ and ಈ
- ಉ and ಊ

Take your time and notice the difference in both shape and pronunciation.

---

## Ignoring ಋ

Although **ಋ** is less common than the other vowels, it still appears in many traditional and Sanskrit-derived words.

It is worth learning early.

# Did You Know?

Unlike English, Kannada has a nearly one-to-one relationship between letters and sounds.

Once you learn the vowels, you'll be able to pronounce many Kannada words correctly simply by reading them.

---

# Teacher's Note

Don't worry if some vowels look similar at first.

Recognition improves naturally with repetition.

Spend just a few minutes each day reading the vowels aloud before moving on to the next chapter.

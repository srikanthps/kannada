# Purpose

This chapter provides a complete Kagunita reference for the Kannada alphabet.

Use it to:

- Check any consonant-vowel combination
- Revise Kagunita
- Confirm spelling while reading or writing
- Practise complete Kagunita rows

The vowel order is the same for every consonant.

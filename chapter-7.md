- Recognize repeating vowel patterns.
- Compare different consonants using the same vowel.

# Review

In the previous chapter, you learned that every consonant follows the same Kagunita pattern.

For example:

| Vowel | ಕ | ಗ | ತ | ಮ |
|--------|---|---|---|---|
| ಅ | ಕ | ಗ | ತ | ಮ |
| ಆ | ಕಾ | ಗಾ | ತಾ | ಮಾ |
| ಇ | ಕಿ | ಗಿ | ತಿ | ಮಿ |
| ಈ | ಕೀ | ಗೀ | ತೀ | ಮೀ |
| ಉ | ಕು | ಗು | ತು | ಮು |
| ಊ | ಕೂ | ಗೂ | ತೂ | ಮೂ |

Notice that **only the consonant changes**.

The vowel pattern stays exactly the same.

# Common Mistakes

## Memorizing Instead of Understanding

Don't think:

> ಕಾ is a separate symbol.

Instead think:

> ಕ + ಾ = ಕಾ

The same rule applies everywhere.

---

## Ignoring the Pattern

Every consonant uses the same vowel signs.

Once you understand one row, you understand every row.

---

# Did You Know?

There are hundreds of possible Kannada syllables, but almost all of them are formed by applying the same Kagunita pattern to different consonants.

Learning the pattern is far easier than memorizing each syllable individually.

---

# Teacher's Note

Spend five minutes each day reading one row and one column.

For example:

- Monday → ಕ
- Tuesday → ಗ
- Wednesday → ತ
- Thursday → ಮ

Within a few weeks, you'll begin recognizing Kagunita without consciously thinking about the vowel signs.

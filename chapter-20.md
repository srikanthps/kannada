# Purpose

This chapter provides a quick reference for the most commonly used Kannada Ottaksharas.

Use it to:

- Identify unfamiliar conjunct consonants.
- Revise common Ottaksharas.
- Improve reading fluency.
- Check the pronunciation of conjuncts.

# Reading Tips

When you encounter an unfamiliar Ottakshara:

1. Identify the first consonant.
2. Identify the second consonant.
3. Blend them together.
4. Read the complete Akshara.
5. Read the whole word again smoothly.

With practice, you will begin recognizing common Ottaksharas instantly.

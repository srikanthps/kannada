# What is Kagunita?

**Kagunita (ಕಾಗುಣಿತ)** is the combination of:

- a consonant (**ವ್ಯಂಜನ**)
- and a vowel sign (**ಮಾತ್ರೆ**)

For example:

```
ಕ + ಾ = ಕಾ
```

The consonant changes its pronunciation depending on the vowel sign that is attached.

Every Kannada consonant follows the same pattern.

---

# The Kagunita Pattern

Let's begin with the consonant **ಕ**.

| Vowel | Akshara | Read As |
|-------|---------|----------|
| ಅ | ಕ | ka |
| ಆ | ಕಾ | kaa |
| ಇ | ಕಿ | ki |
| ಈ | ಕೀ | kii |
| ಉ | ಕು | ku |
| ಊ | ಕೂ | kuu |
| ಋ | ಕೃ | kru |
| ಎ | ಕೆ | ke |
| ಏ | ಕೇ | kee |
| ಐ | ಕೈ | kai |
| ಒ | ಕೊ | ko |
| ಓ | ಕೋ | koo |
| ಔ | ಕೌ | kau |
| ಅಂ | ಕಂ | kam |
| ಅಃ | ಕಃ | kah |

Read the entire row aloud several times.

# One Pattern, Many Consonants

Once you understand the Kagunita pattern, you can apply it to every consonant.

For example:

| Base Letter | Long "aa" Form |
|-------------|----------------|
| ಕ | ಕಾ |
| ಗ | ಗಾ |
| ತ | ತಾ |
| ದ | ದಾ |
| ಪ | ಪಾ |
| ಮ | ಮಾ |

Similarly:

| Base Letter | "i" Form |
|-------------|-----------|
| ಕ | ಕಿ |
| ಗ | ಗಿ |
| ತ | ತಿ |
| ದ | ದಿ |
| ಪ | ಪಿ |
| ಮ | ಮಿ |

You are learning a pattern—not individual symbols.

# Common Mistakes

## Trying to Memorize Every Row

You do **not** need to memorize separate Kagunita tables for every consonant.

Learn the pattern once.

Then apply it everywhere.

---

## Forgetting the Inherent Vowel

Remember:

```
ಕ
```

already means **ka**.

It does **not** mean just **k**.

---

## Reading the Vowel Before the Consonant

Even when a vowel sign appears before the consonant (such as **ಕೆ**), always read:

```
ಕೆ = ke
```

Never:

```
ek
```

# Did You Know?

Kannada has dozens of consonants, but they all use the **same set of vowel signs**.

That means once you understand the Kagunita pattern, you can read thousands of Kannada syllables without learning a new rule.

---

# Teacher's Note

Don't rush through this chapter.

Kagunita is the bridge between learning individual letters and reading complete words.

Spend a few minutes each day reading one or two Kagunita rows aloud.

Once the pattern becomes familiar, reading Kannada becomes much easier.

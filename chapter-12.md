> *"Words name things. Sentences express ideas."*

# What You Will Learn

In this chapter, you will:

- Read simple Kannada sentences.
- Improve reading fluency.
- Recognize familiar words in context.
- Read short paragraphs.
- Build confidence before reading stories.

This chapter introduces very little new vocabulary. Instead, it focuses on using the words you have already learned.

---

# How to Read a Sentence

When reading a sentence:

1. Read one word at a time.
2. Pause briefly at spaces.
3. Read the complete sentence again smoothly.
4. Think about its meaning.

Don't worry if your first reading is slow.

Read these simple sentences.

| Kannada | Transliteration | Meaning |
|----------|-----------------|---------|
| ಇದು ಮನೆ. | idu mane. | This is a house. |
| ಇದು ಆನೆ. | idu aane. | This is an elephant. |
| ಇದು ಪುಸ್ತಕ. | idu pustaka. | This is a book. |

# Reading Practice

Read the following paragraph.

```
ಇದು ನಮ್ಮ ಮನೆ.

ಅಮ್ಮ ಪುಸ್ತಕ ಓದುತ್ತಾರೆ.

ಅಪ್ಪ ಬರೆಯುತ್ತಾರೆ.

ಅಕ್ಕ ಶಾಲೆಗೆ ಹೋಗುತ್ತಾಳೆ.

ನಾನು ಮನೆಯಲ್ಲಿ ಓದುತ್ತೇನೆ.
```

Now read it again without stopping after every word.

---

# Comprehension Check

Answer these questions.

1. ಇದು ಯಾರ ಮನೆ?
2. ಯಾರು ಪುಸ್ತಕ ಓದುತ್ತಾರೆ?
3. ಅಕ್ಕ ಎಲ್ಲಿಗೆ ಹೋಗುತ್ತಾಳೆ?
4. ಯಾರು ಮನೆಯಲ್ಲಿ ಓದುತ್ತಾರೆ?

Try answering in English first.

---

# Reading Challenge

Read the following without looking at the transliteration.

```
ಇದು ನಮ್ಮ ಮನೆ.

ಅಮ್ಮ ಓದುತ್ತಾರೆ.

ಅಪ್ಪ ಬರೆಯುತ್ತಾರೆ.

ಮಕ್ಕಳು ಆಡುತ್ತಾರೆ.
```

Repeat until you can read smoothly.

---

# Common Mistakes

## Reading Every Word Separately

Instead of:

> ಇದು ... ನನ್ನ ... ಮನೆ

Try reading:

> ಇದು ನನ್ನ ಮನೆ.

as one smooth sentence.

---

## Ignoring Punctuation

A full stop (.) tells you where to pause.

Pausing at the correct place makes reading much easier.

# Did You Know?

Children learning Kannada usually begin reading simple sentences long before they understand every grammar rule. Repeated reading helps them recognize common sentence patterns naturally.

---

# Teacher's Note

Don't worry if some sentences feel difficult.

Read each sentence:

1. Slowly.
2. Smoothly.
3. Naturally.

After a few days of practice, you'll notice that familiar words become much easier to recognize.

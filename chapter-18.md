# Purpose

This chapter is intended as a quick reference.

Use it whenever you need to:

- Check the Kannada alphabet
- Review vowels or consonants
- Confirm the traditional alphabetical order
- Revise before moving on to Kagunita or Ottakshara

# Complete Kannada Alphabet

```text
ಸ್ವರಗಳು

ಅ ಆ ಇ ಈ ಉ ಊ ಋ ಎ ಏ ಐ ಒ ಓ ಔ ಅಂ ಅಃ

ವ್ಯಂಜನಗಳು

ಕ ಖ ಗ ಘ ಙ

ಚ ಛ ಜ ಝ ಞ

ಟ ಠ ಡ ಢ ಣ

ತ ಥ ದ ಧ ನ

ಪ ಫ ಬ ಭ ಮ

ಯ ರ ಲ ವ

ಶ ಷ ಸ ಹ

ಳ ಕ್ಷ ಜ್ಞ
```

# Daily Revision

Spend two or three minutes each day:

1. Read the vowels.
2. Read each consonant group.
3. Read the complete alphabet once.
4. Identify any letters you still find difficult.
5. Repeat the next day.

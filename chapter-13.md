# Passage 1 – ನಮ್ಮ ಮನೆ (Our House)

## Kannada

```
ಇದು ನಮ್ಮ ಮನೆ.

ನಮ್ಮ ಮನೆಯಲ್ಲಿ ಅಮ್ಮ, ಅಪ್ಪ ಮತ್ತು ಅಕ್ಕ ಇದ್ದಾರೆ.

ಮನೆ ಮುಂದೆ ಒಂದು ಮರ ಇದೆ.

ಮರದಲ್ಲಿ ಹಕ್ಕಿಗಳು ಕುಳಿತಿವೆ.

ನನಗೆ ನಮ್ಮ ಮನೆ ತುಂಬ ಇಷ್ಟ.
```

## Glossary

| Kannada | Meaning |
|----------|---------|
| ಮನೆ | House |
| ಮರ | Tree |
| ಹಕ್ಕಿ | Bird |
| ಮುಂದೆ | In front of |
| ತುಂಬ | Very |
| ಇಷ್ಟ | Like |

---

## Questions

1. ಮನೆಯಲ್ಲಿ ಯಾರು ಇದ್ದಾರೆ?
2. ಮನೆ ಮುಂದೆ ಏನು ಇದೆ?
3. ಮರದಲ್ಲಿ ಏನು ಕುಳಿತಿವೆ?
4. ಯಾರಿಗೆ ಮನೆ ಇಷ್ಟ?

# Passage 2 – ಶಾಲೆಯ ದಿನ (A Day at School)

## Kannada

```
ಇಂದು ನಾನು ಶಾಲೆಗೆ ಹೋದೆ.

ನನ್ನ ಬಳಿ ಪುಸ್ತಕ ಮತ್ತು ಪೆನ್ಸಿಲ್ ಇವೆ.

ಶಿಕ್ಷಕರು ಹೊಸ ಪಾಠ ಹೇಳಿದರು.

ನಾನು ಚೆನ್ನಾಗಿ ಓದೆ.

ಸಂಜೆ ಮನೆಗೆ ಬಂದೆ.
```

## Glossary

| Kannada | Meaning |
|----------|---------|
| ಇಂದು | Today |
| ಶಾಲೆ | School |
| ಪುಸ್ತಕ | Book |
| ಪೆನ್ಸಿಲ್ | Pencil |
| ಶಿಕ್ಷಕರು | Teacher |
| ಪಾಠ | Lesson |
| ಸಂಜೆ | Evening |

---

## Questions

1. ನಾನು ಎಲ್ಲಿಗೆ ಹೋದೆ?
2. ನನ್ನ ಬಳಿ ಏನು ಇವೆ?
3. ಯಾರು ಪಾಠ ಹೇಳಿದರು?
4. ನಾನು ಯಾವಾಗ ಮನೆಗೆ ಬಂದೆ?

# Passage 3 – ತೋಟದಲ್ಲಿ (In the Garden)

## Kannada

```
ನಮ್ಮ ತೋಟದಲ್ಲಿ ಹಲವಾರು ಹೂವುಗಳಿವೆ.

ಕೆಂಪು, ಹಳದಿ ಮತ್ತು ಬಿಳಿ ಹೂವುಗಳು ಅರಳಿವೆ.

ಜೇನುನೊಣ ಹೂವಿನ ಮೇಲೆ ಕುಳಿತಿದೆ.

ಚಿಟ್ಟೆ ಹಾರುತ್ತಿದೆ.

ತೋಟ ಬಹಳ ಸುಂದರವಾಗಿದೆ.
```

## Glossary

| Kannada | Meaning |
|----------|---------|
| ತೋಟ | Garden |
| ಹೂವು | Flower |
| ಕೆಂಪು | Red |
| ಹಳದಿ | Yellow |
| ಬಿಳಿ | White |
| ಜೇನುನೊಣ | Honeybee |
| ಚಿಟ್ಟೆ | Butterfly |
| ಸುಂದರ | Beautiful |

---

## Questions

1. ತೋಟದಲ್ಲಿ ಏನುಗಳಿವೆ?
2. ಹೂವುಗಳು ಯಾವ ಬಣ್ಣದಲ್ಲಿವೆ?
3. ಹೂವಿನ ಮೇಲೆ ಏನು ಕುಳಿತಿದೆ?
4. ತೋಟ ಹೇಗಿದೆ?

---

# Reading Practice

Read each passage three times.

### First Reading

Read slowly and carefully.

### Second Reading

Read more smoothly.

### Third Reading

Try to understand the meaning without looking at the glossary.

---

# Reading Tips

- Read aloud whenever possible.
- Pause at full stops.
- Do not stop after every word.
- Try to understand the complete sentence.
- Re-read difficult passages.

Remember: reading becomes easier through repetition.

---

# Challenge

After reading all three passages:

- Can you identify every vowel sign?
- Can you recognize Ottaksharas?
- Can you spot Arkavattu?
- Can you explain the passage in English?

If yes, you're making excellent progress.

---

# Did You Know?

Children become fluent readers by reading many short passages repeatedly, not by memorizing long word lists. Reading familiar vocabulary in different contexts helps your brain recognize words automatically.

---

# Teacher's Note

Don't worry if you don't understand every word immediately.

A good reading habit is:

1. Read the passage.
2. Read it again.
3. Check unfamiliar words.
4. Read it one more time.

You'll notice that each reading becomes easier than the previous one.

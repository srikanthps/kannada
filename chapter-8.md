# What is an Ottakshara?

An **Ottakshara (ಒತ್ತಕ್ಷರ)** is a **conjunct consonant**.

It is formed when **two or more consonants occur together without an intervening vowel**.

Instead of writing the consonants separately, Kannada combines them into a single written unit (Akshara).

---

# Why Do We Need Ottakshara?

Suppose we want to write the sound **kka**.

Writing:

```
ಕಕ
```

would be read as:

> ka + ka

Instead, Kannada removes the vowel **ಅ** from the first consonant and joins it with the second consonant.

```
ಕ್ + ಕ = ಕ್ಕ
```

This new combined form is called an **Ottakshara**.

---

# How is an Ottakshara Formed?

The basic process is:

1. Take the first consonant.
2. Remove its inherent vowel **ಅ** using the virama (್).
3. Join it with the next consonant.

Examples:

| Combination | Ottakshara | Read As |
|------------|------------|---------|
| ಕ್ + ಕ | ಕ್ಕ | kka |
| ಪ್ + ಪ | ಪ್ಪ | ppa |
| ಮ್ + ಮ | ಮ್ಮ | mma |
| ತ್ + ತ | ತ್ತ | tta |
| ಲ್ + ಲ | ಲ್ಲ | lla |

---

# Common Double Consonants

These Ottaksharas appear frequently in everyday Kannada.

| Ottakshara | Example | Meaning |
|------------|---------|---------|
| ಕ್ಕ | ಅಕ್ಕ | elder sister |
| ಪ್ಪ | ಅಪ್ಪ | father |
| ಮ್ಮ | ಅಮ್ಮ | mother |
| ಲ್ಲ | ಕಲ್ಲು | stone |
| ನ್ನ | ಸಣ್ಣ | small |

Read each word aloud.

---

# Ottaksharas with Different Consonants

Not all Ottaksharas are formed from identical consonants.

Examples:

| Ottakshara | Example | Meaning |
|------------|---------|---------|
| ಕ್ರ | ಕ್ರಿಕೆಟ್ | cricket |
| ಗ್ರ | ಗ್ರಾಮ | village |
| ಪ್ರ | ಪ್ರಾಣಿ | animal |
| ತ್ರ | ಪತ್ರ | letter |
| ಬ್ರ | ಬ್ರಹ್ಮ | Brahma |

You will encounter these combinations frequently while reading Kannada.

# Virama (್)

The small mark below a consonant is called the **Virama (್)**.

It removes the inherent vowel **ಅ** from a consonant.

For example:

```
ಕ  →  ka

ಕ್ →  k
```

This allows the consonant to join another consonant and form an Ottakshara.

---

# Common Mistakes

## Reading Each Consonant Separately

Incorrect:

```
ಕ್ ... ಕ
```

Correct:

```
ಕ್ಕ
```

Read it as one unit.

---

## Ignoring the Ottakshara

When you see:

```
ಗ್ರ
```

Don't read:

> ಗ ರ

Read:

> gra

Treat the conjunct as a single Akshara.

# Did You Know?

Ottaksharas allow Kannada to represent consonant clusters clearly and compactly.

Many Indian scripts—including Kannada, Telugu, Malayalam, Devanagari, and Bengali—use conjunct consonants as part of their writing system.

---

# Teacher's Note

At first, Ottaksharas may look like completely new letters.

They're not.

Each one is simply two (or sometimes more) consonants joined together.

As you continue reading Kannada, you'll begin recognizing the most common Ottaksharas instantly.
